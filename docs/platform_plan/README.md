- **프론트엔드**: Next.js 15, React 19, TanStack Query, Zustand
- **백엔드**: NestJS, Drizzle ORM, PostgreSQL
- **스타일**: Tailwind CSS 4, Radix UI
- **기타**: Biome (린팅), Turborepo (모노레포)
